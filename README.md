## Technology Stack

This website is built with modern web technologies to ensure a fast, responsive, and engaging user experience:

- **Frontend Framework**: React 18 with Vite for lightning-fast development
- **Styling**: Tailwind CSS for a premium, responsive design
- **Animations**: Framer Motion for smooth, interactive UI elements
- **3D Graphics**: Three.js (React Three Fiber) for immersive visual experiences
- **Icons**: Lucide React for clean, modern iconography
- **Routing**: React Router DOM for seamless navigation
- **Analytics**: Google Analytics for insights and optimization
- **Deployment**: Vercel for reliable, global CDN hosting

## Local Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The development server will start at `http://localhost:5173/`
